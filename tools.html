Who this is for

For operations managers who prefer to run improvements independently.

The toolkit is a self-serve option for operations managers who know what they need to track but don't have time to build the tracking infrastructure from scratch.

Each template is built from the same tools used during PainKiller audits — formats that have been refined across multiple operations and sectors.

If you'd prefer someone else to do the diagnostics, the On-Site Audit is the right place to start.

Toolkit vs Audit — which is right?

Choose the toolkit if: you have a specific tracking gap, you prefer to run improvements in-house, or you want a low-cost starting point before considering consultancy.

Choose the audit if: you've already tried internal fixes and the problem keeps coming back, or you need an independent view of where the operation is losing value.
